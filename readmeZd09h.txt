    Задание 09h  Поиск пути в матрице.
    ==================================
    
   1. Матрица  А задает поле размером  n*n, каждая клетка
   которого определяет  высоту местности,  заданную целым
   числом от 0 до 99.     Из  верхнего  углового  поля  с 
   координатами  (1,1) бьет источник и  вода из него рас-
   пространяется на соседнии клетки  (соседними  клетками
   для каждой клетки считаем 8 окружающих ее клеток, если 
   клетка не на границе матрицы) при условии:   высота  в 
   соответствующей клетке не превосходит высоту местности
   в  текущей  клетке, в которую вода уже попала,т.е.
   высота, заданная в клетке,  в которую вода перетекает,
   должна быть <= высоты клетки, из которой она течет.
      Определить, может ли вода из источника при этих ус-
   виях достигнуть поля с координатами (n,n) и,  если мо-
   жет, то вывести один из возможных путей  течения воды.
      Реализовать рекурсивную процедуру, решающую эту за-
   дачу, используя алгоритм поиска в глубину (DFS -Depth
   First Search). При этом считать, что 
      а) n<=10;
      б) исходная матрица задана текстовым файлом,  в ко-
         тором  в  первой строке задано значение  n,  а в
         каждой следующей  -  построчно (через пробел(ы)) 
         заданы элементы этой матрицы (n строк);
      в) протестировать полученное решение в частности  и
         на прилагаемых исходных данных в файле plan.txt.
         
   2. Реализовать диалог при выполнении программы,  в ко-
   тором бы запрашивалось имя файла  с исходными данными.
   Обеспечить возможность (при одном запуске программы на
   выполнение) решить задачу для нескольких файлов  с ис-
   ходными данными.
   
   3. Создать  несколько  файлов с исходными данными  для
   разных значений n <= 10 с положительными и отрицатель-
   ными результатами поиска пути в матрице.
