Задание 12h
    ===========
    
 1. На шахматной доске размером n на n (n<=20),
    расставить n ферзей так, чтобы они 
    не били друг друга, то есть, каждая
    горизонталь, вертикаль и любая диагональ
    шахматной доски должна содержать ровно 
    одного ферзя, например, для n=4 имеем:
         0 х 0 0
         0 0 0 х
         х 0 0 0
         0 0 х 0
   (здесь литерой х обозначено местоположение ферзя)
   
 2. Если такая расстановка невозможна (например, для
    n=2 или n=3), то выдать соответствующее сообщение.
    
 3. Если таких расстановок несколько, то реализовать:
     а) выдачу только первой найденной расстановки
         (queen1_nn.exe, nn -Ваш суффикс);
     б) последовательную выдачу (по запросу) всех
        возможных расстановок (queen2_nn.exe).
      
 4. Решение задачи должно быть реализовано в виде
    рекурсивной процедуры или функции.
